Ruby 快速筆記

2015-04-04
Program
ruby

Ruby 快速筆記

  • Ruby 中的字串需要包在 " 或者 ' 中不過兩者有一點微妙的差別，在雙引號中可以使用逸出字元(\n \r \) 等等，不過單引號會原封不動的輸出除了 \\ 例外
  • 在呼叫方法的時候 Ruby 可以省略 ()
  • puts 和 print 的差別是 puts 會在行末加上 \n，如果是用 , 分開不同的參數輸出則每個參數後面都會有 \n
  • 第三個輸出的 method 是 p ，puts 和 print 不管輸出的是數字或字串都只會顯示內容
