Atom 偏好設定

Use Preview Tabs

說明

點擊檔案一下為預覽檔案，兩下才開啟分頁

設定

  1. 開啟 Preferences
  2. 選取 Packages
  3. 搜尋 tabs
  4. 點 Settings
  5. 勾選 Use Preview Tabs

關閉自動補齊加入 Buffers

說明

自動補齊會把之前打過的字加入，很惱人。

設定

  1. 開啟 Preferences
  2. 選取 Packages
  3. 搜尋 autocomplete-plus
  4. 點 Settings
  5. 關閉 Include Completions From All Buffers
